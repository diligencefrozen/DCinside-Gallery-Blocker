게시글의 핵심 부분만 빠르게 확인합니다.

게시물 목록 제목, 게시물 제목, 본문, 댓글에 포함된 키워드를 감지해 완전히 차단합니다.

    완전히 막지 않고 접어둔 뒤, 필요하면 “계속 보기”로 열 수 있습니다.

      기존 키워드 차단과 달리, 차단된 항목을 “계속 보기”로 다시 열 수 있습니다.

      최소 10초, 최대 600초(10분)까지 설정 가능합니다.

      초보 모드에서 카운트다운 후 리다이렉트되는 시간입니다.

      숨길 영역은 설정 메뉴 → “메인 페이지 영역 숨기기”에서 관리하세요.

      숨길 영역은 설정 메뉴 → “갤러리 페이지 영역 숨기기”에서 관리하세요.

      숨길 영역은 설정 메뉴 → “검색 페이지 영역 숨기기”에서 관리하세요.

      TIP 빠른 이용자 차단

      작성자 닉네임, 갤로그 아이콘, 메모 버튼 위에서 우클릭하면 바로 차단됩니다.

      글 제목이나 본문을 우클릭하면 기존 브라우저 메뉴가 열립니다.

      닉네임 게임메카 또는 UID gamemeca로 작성된 글과 댓글을 숨깁니다.

      닉네임 댓글돌이, dory 댓글, issuefeed형 댓글돌이 광고 댓글을 숨깁니다.

      목록에서 AD 또는 설문 딱지가 붙은 운영자의 글을 숨깁니다.

      게시물 목록을 더 촘촘하게 표시합니다. UID와 메모 버튼은 목록에서 작게 정리됩니다.

        * UID는 작성자 요소의 data-uid 값 또는 갤로그 주소의 사용자 ID입니다.
        * 비회원은 IP 앞부분(예: 39.7)을 등록하면 일괄 차단됩니다.

        디시 메모와 무관하게 확장프로그램 내부에만 저장됩니다.
