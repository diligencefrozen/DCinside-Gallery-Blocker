디시갤 차단기 설정

차단할 갤러리와 숨길 영역을 손쉽게 관리하세요.

🔤 글꼴 설정

기본값은 Noto Sans Korean / Noto Sans KR Regular 400입니다. 선택한 글꼴은 options, popup, 디시인사이드 페이지에 함께 적용됩니다.

100%
디시 페이지에도 적용 기본값: Noto Sans Korean Regular 400
Google Fonts 한글 글꼴 둘러보기 ↗

📦 설정 백업/복원

현재 저장된 설정을 JSON 파일로 내보내거나 불러올 수 있습니다.

불러오면 기존 설정이 덮어씌워집니다. (chrome.storage.sync 기준)

🚧 갤러리 접근 차단

차단 갤러리에 들어갔을 때 어떤 방식으로 막을지 설정합니다. popup의 “갤러리 차단”과 같은 설정입니다.

갤러리 차단 등록된 차단 갤러리 접근을 막습니다.
차단 방식 차단 방식을 선택하세요.

리다이렉트 지연 시간

초보 모드에서 카운트다운 후 메인으로 이동하기까지 기다리는 시간입니다.

🔄 자동 새로고침

갤러리 목록을 일정 간격으로 새로고침합니다. 본문/댓글을 읽는 중에는 방해를 줄이도록 일시중지됩니다.

자동 새로고침 현재 페이지를 설정한 간격마다 갱신합니다.

최소 10초, 최대 600초(10분)까지 설정 가능합니다.

🧩 화면/표시 빠른 설정

자주 쓰는 표시 옵션을 한 곳에 모았습니다. 숨길 영역의 세부 CSS 선택자는 아래 영역 숨기기 섹션에서 관리합니다.

메인 페이지 영역 숨김 www.dcinside.com의 불필요한 영역을 숨깁니다.
갤러리 페이지 영역 숨김 갤러리 목록/게시글의 지정 영역을 숨깁니다.
검색 페이지 영역 숨김 search.dcinside.com의 지정 영역을 숨깁니다.
닉네임 옆 회원 ID 표시 작성자 닉네임 옆에 UID 배지를 표시합니다.
컴팩트 모드 게시물 목록을 더 촘촘하게 정리합니다.
이용자 메모 확장프로그램 내부 메모 표시 기능을 켜거나 끕니다.

🖥️ 페이지 미리보기

게시글을 새 탭 없이 오버레이로 미리봅니다. 사용하고싶지 않으면 끌 수 있습니다.

🔎 키워드 차단 모드

게시물 목록 제목, 게시물 제목, 본문, 댓글에 포함된 키워드를 감지해 완전히 차단합니다.

이미지 안에 박힌 글자는 텍스트가 아니므로 감지하지 않습니다.

차단 대상

차단 키워드 추가

등록한 키워드는 대소문자와 공백 차이를 정리해서 비교합니다.

차단 키워드 리스트

    👁️ 키워드 숨기기 모드

    숨길 키워드가 포함된 글이나 댓글을 접어둡니다. 숨겨진 항목은 “계속 보기”로 다시 열 수 있습니다.

    기존 키워드 차단 기능과 별도로 동작합니다.

    차단 대상

    차단 키워드 추가

    쉼표로 여러 차단 키워드를 한 번에 추가할 수 있습니다. 예: 단어1, 단어2

    차단 키워드 리스트

      기존 키워드 차단과 달리, 이 모드에서는 차단된 항목을 “계속 보기”로 열 수 있습니다.

      사용자 차단

      ↗

      작성자 위에서 우클릭하면 즉시 차단됩니다.

      닉네임, 갤로그 아이콘, 메모 버튼 위에서 우클릭하세요. 차단 대상은 자동으로 UID 또는 IP로 저장됩니다.

      실수 방지를 위해 글 제목, 본문, 댓글 내용 영역에서는 즉시 차단이 실행되지 않습니다.

      OFF면 UID(또는 UIP)가 등록되어 있어도 차단하지 않습니다.

        * 유저 아이디(UID)는 작성자 요소의 data-uid 값 또는 갤로그 주소의 사용자 ID입니다.

        * 비회원은 IP 앞부분(예: 39.7)을 등록하면 일괄 차단됩니다.

        직접 갤러리 ID 추가

          🛡️ 추천 차단 갤러리

          악성 게시물이 자주 업로드된 갤러리입니다.

            🧹 메인 페이지 영역 숨기기

            디시인사이드 메인에서 제거할 영역을 지정하세요.
            예: div.time_best

              🧹 갤러리 페이지 영역 숨기기

              갤러리(게시글,목록) 페이지에서 숨길 영역을 지정하세요.
              예: div.ad_bottom_list

                🧹 검색 페이지 영역 숨기기

                검색 페이지에서 숨길 영역을 지정하세요.
                예: div.integrate_cont.news_result

                  👻 비회원 게시물 숨기기

                  갤로그 링크가 없는 비회원이 작성한 글과 댓글을 숨깁니다.

                  🎮 게임메카 글/댓글 숨기기

                  닉네임 게임메카 또는 UID gamemeca로 작성된 글과 댓글을 숨깁니다.

                  🤖 댓글돌이 광고 숨기기

                  닉네임 댓글돌이, dory 댓글, issuefeed형 댓글돌이 광고 댓글을 숨깁니다.

                  예: <span class="gall_writer" data-nick="댓글돌이">, <li class="ub-content dory">

                  📢 운영자 광고, 설문 글 숨기기

                  목록에서 AD 또는 설문 딱지가 붙은 운영자의 글을 숨깁니다.

                  예: <td class="gall_subject"><b>AD</b></td>, <b>설문</b>

                  💬 댓글 숨기기

                  일반 댓글 숨기기
                  이미지 댓글 숨기기
                  디시콘 숨기기

                  * 일반 댓글: 게시글 하단의 댓글 영역
                  * 이미지 댓글: 이미지에 달린 댓글 영역만 숨김
                  * 디시콘: 본문과 댓글의 디시콘 숨김

                  📝 이용자 메모 관리

                  회원은 아이디 기준, 비회원은 아이피 기준으로 저장됩니다.
