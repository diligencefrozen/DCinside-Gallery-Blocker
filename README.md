# DCinside Gallery Blocker

Less noise. More focus.

DCinside Gallery Blocker is a lightweight Chrome extension that helps you clean up your DCinside experience by hiding unwanted galleries, posts, comments, users, images, and DCCons.

It works locally, only on DCinside pages, and does not communicate with external servers.

## Key Features

### Gallery Blocking

Block access to galleries you do not want to see.

### Real-Time Best Hiding

Hide distracting Real-Time Best content and keep your screen clean.

### Post, Comment, and User Filtering

Filter unwanted posts, comments, users, images, and DCCons.

### Automatic Cleanup

Your settings stay active after page refreshes, so the screen remains clean without extra effort.

## Recommended For

- Users who want to browse DCinside mainly for information
- Users who want to reduce exposure to provocative content
- Users who prefer a cleaner, less cluttered interface
- Users who want a more focused digital environment

## Privacy-Friendly Design

### Local Only

The extension does not communicate with external servers.

### DCinside Only

It only runs on DCinside-related pages.

### Lightweight

Built to stay fast, simple, and focused.

## Installation

[**Install from the Chrome Web Store**](https://chromewebstore.google.com/detail/디시갤-차단기/fnfmdbldnhadkadklplhcjcojjiaopgg)

1. Click **Add to Chrome**
2. Click the 🛡️ icon next to the address bar
3. Open **Settings**
4. Configure blocked galleries, delay time, comment hiding, and other options

> **Tip:** Smart Blocking mode is strongly recommended.
