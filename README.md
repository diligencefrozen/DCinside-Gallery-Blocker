<h2>디시갤 차단기는</h2>
<pre>
디시 화면에서 불필요한 요소를 조용히 제거합니다.
눈에 거슬리는 콘텐츠 없이,
보고 싶은 것만 남깁니다.
</pre>

<h1>핵심 기능</h1>
<h2>갤러리 차단</h2>
원하지 않는 갤러리 접근을 완전히 차단합니다.

<h2>실시간 베스트 숨김</h2>
자극적인 실베 노출 없이 깔끔한 디시 화면을 유지합니다.

<h2>게시글,댓글,유저 필터링</h2>
특정 유저, 댓글, 이미지, 디시콘을 보이지 않게 처리합니다.

<h2>자동 정리</h2>
<p>페이지가 새로 고쳐져도 설정은 유지됩니다.</p>
따로 신경 쓸 필요가 없습니다.

<h1>이런 분께 추천합니다</h1>
<h2>디시를 정보 위주로만 보고 싶은 분</h2>

<h2>자극적인 콘텐츠 노출을 줄이고 싶은 분</h2>

<h2>화면이 복잡한 웹사이트가 피로하게 느껴지는 분</h2>

<h2>디지털 환경을 미니멀하게 유지하고 싶은 분</h2>

<h1>설계 철학</h1>
<h2>로컬에서만 동작</h2>
외부 서버와 통신하지 않습니다.

<h2>필요한 것만 접근</h2>
디시 페이지에서만 작동합니다.

<h2>가볍고 빠르게</h2>
불필요한 기능 없이, 핵심만 담았습니다.

<h1>한 줄 요약</h1>
<pre>
디시는 그대로, 화면만 정리합니다.
당신의 디시 사용 방식을 바꾸세요.
불필요한 것은 사라집니다.
</pre>
